1. Como un nuevo usuario, quiero ver una pantalla de inicio animada para que tenga una introducción atractiva a la app antes de iniciar sesión.

2. Como un usuario, quiero registrarme con mi email, contraseña y nombre para que pueda crear una nueva cuenta.

3. Como un usuario, quiero verificar mi email después del registro para que mi cuenta esté segura y pueda continuar.

4. Como un nuevo usuario de Google sign-in, quiero seleccionar mi rol (estudiante o profesor) para que la app configure la interfaz para mis necesidades.

5. Como un usuario, quiero iniciar sesión usando email/contraseña o Google Sign-In para que pueda acceder a mi panel personalizado.

6. Como profesor, quiero recibir automáticamente un código único al primer inicio de sesión para que pueda compartirlo con estudiantes para conectarme.

7. Como estudiante, quiero agregar profesores usando su código único para que pueda acceder a sus quizzes públicos.

8. Como profesor, quiero crear quizzes con título, descripción e imagen opcional para que pueda diseñar contenido educativo.

9. Como profesor, quiero configurar quizzes como públicos o privados para que controle la visibilidad a estudiantes.

10. Como usuario, quiero editar quizzes existentes (título, descripción, imagen, visibilidad) para que pueda actualizar contenido según sea necesario.

11. Como usuario, quiero agregar preguntas a quizzes con un enunciado y cuatro opciones (una correcta) para que los quizzes sean interactivos y evalúen conocimientos.

12. Como usuario, quiero editar o eliminar preguntas en mis quizzes para que pueda mantener contenido preciso.

13. Como estudiante, quiero ver una lista de quizzes disponibles de mis profesores agregados para que pueda elegir qué estudiar.

14. Como usuario, quiero ver una lista de mis propios quizzes creados para que pueda gestionarlos.

15. Como estudiante, quiero realizar quizzes respondiendo preguntas secuencialmente para que pueda probar mi conocimiento.

16. Como estudiante, quiero ver resultados detallados después de completar un quiz (puntuación, respuestas correctas/incorrectas) para que pueda aprender de la retroalimentación.

17. Como profesor, quiero ver una lista de estudiantes que me han agregado para que pueda rastrear mi clase.

18. Como estudiante, quiero ver y gestionar mi lista de profesores agregados para que pueda organizar mis conexiones.

19. Como usuario, quiero navegación inferior basada en roles (Inicio para quizzes, Puntuación para resultados, lista de Estudiantes/Profesores) para que acceda a características relevantes eficientemente.

20. Como usuario, quiero que la app verifique autenticación al inicio y reanudación de la app para que la seguridad se mantenga.

=== LECTURAS EN FIREBASE ===

1. MainActivity.java:
   - db.collection("usuarios").document(userId).get() - Obtener rol del usuario al iniciar la app
   - db.collection("usuarios").document(userId).get() - Verificar si profesor tiene código asignado

2. CrearQuizActivity.java:
   - FirebaseFirestore.getInstance().collection("usuarios").document(userId).get() - Obtener rol del usuario para configurar interfaz
   - db.collection("quizzes").document(quizIdEdit).get() - Cargar estado de esPublico al editar quiz

3. TeacherFragment.java:
   - db.collection("usuarios").document(currentUser.getUid()).addSnapshotListener() - Escuchar cambios en profesores agregados del estudiante
   - db.collection("usuarios").document(profesorId).get() - Obtener datos de cada profesor agregado
   - db.collection("quizzes").whereIn("userId", profesoresIds).whereEqualTo("esPublico", true).orderBy("fechaCreacion", Query.Direction.DESCENDING).addSnapshotListener() - Escuchar quizzes públicos de profesores agregados

4. StudentFragment.java:
   - db.collection("usuarios").whereArrayContains("profesoresAgregados", currentUser.getUid()).whereEqualTo("rol", "estudiante").addSnapshotListener() - Escuchar estudiantes que agregaron al profesor
   - db.collection("usuarios").document(estudianteId).addSnapshotListener() - Escuchar cambios individuales en cada estudiante (para detectar si removieron al profesor)
   - db.collection("quizzes").whereEqualTo("userId", currentUser.getUid()).addSnapshotListener() - Escuchar quizzes del profesor para calcular métricas

5. ScoreFragment.java:
   - db.collection("resultados").whereEqualTo("userId", currentUser.getUid()).orderBy("fecha", Query.Direction.DESCENDING).addSnapshotListener() - Escuchar resultados de quizzes del usuario

6. PreguntasActivity.java:
   - db.collection("quizzes").document(quizId).addSnapshotListener() - Escuchar cambios en datos del quiz (título, descripción, etc.)
   - db.collection("quizzes").document(quizId).collection("preguntas").orderBy("orden").addSnapshotListener() - Escuchar preguntas del quiz

7. ResolverQuizActivity.java:
   - db.collection("quizzes").document(quizId).collection("preguntas").orderBy("orden").get() - Obtener todas las preguntas del quiz para resolverlo

8. HomeFragment.java (asumido por contexto, no leído directamente):
   - Lecturas para mostrar lista de quizzes propios o disponibles

=== ARCHIVOS CON LISTENERS EN TIEMPO REAL ===

1. TeacherFragment.java:
   - profesoresListener: Escucha cambios en profesores agregados del estudiante
   - quizzesListener: Escucha quizzes públicos de profesores agregados

2. StudentFragment.java:
   - estudiantesListener: Escucha estudiantes que agregaron al profesor
   - estudiantesListenersMap: Listeners individuales para cada estudiante (para detectar remociones)
   - quizzesListener: Escucha quizzes del profesor para métricas

3. ScoreFragment.java:
   - resultadosListener: Escucha resultados de quizzes del usuario

4. PreguntasActivity.java:
   - Listener del quiz: Escucha cambios en datos del quiz (título, descripción, etc.)
